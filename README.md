文件夹说明

- assets
  - style
    - common.scss 全局通用样式文件
- components 组件
- hooks 钩子
- router 路由
- store 状态管理
- utils 工具
  - 存放例如 Axios 基础请求, 如像是要请求登陆/注册, 请存放于 hooks 文件夹内.
- views 视图
